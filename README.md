# 📈 股票自动分析程序

基于 **akshare**（国内最稳定免费股票行情库）的全自动股票分析工具，每日一键运行即可生成结构化分析报告。

---

## 功能概览

| 模块 | 功能 |
|------|------|
| 🔍 **自选股跟踪** | 石头科技(688169)、科沃斯(603486)、嘉美包装(002969) 每日行情自动抓取 |
| 📊 **技术指标** | 5/10/20 日均线、MACD、K线多空趋势判断、支撑/压力估算 |
| 🔥 **热门筛选** | 全市场涨幅靠前 + 半导体/光模块板块重点标的筛选 |
| 📋 **操作参考** | 多空信号综合评分 → 观望/持有/加仓/减仓建议（仅供参考！） |
| 🖨️ **控制台报告** | 清晰结构化打印，一目了然 |
| 🌐 **HTML 日报** | 自动生成 `YYYYMMDD-股票分析报告.html` 至 `D:\GUPIAO\` |
| 💾 **CSV 存档** | 每日行情自动追加至本地 CSV，K 线历史独立归档 |

## 自定义配置

编辑 `config.py` 即可修改：

- **自选股**: 修改 `WATCH_STOCKS` 字典（名称→代码）
- **关注板块**: 修改 `HOT_SECTORS`（添加/删除板块及关键词）
- **筛选阈值**: 调整 `min_pct_change`（最低涨幅）、`min_amount`（最低成交额）
- **均线周期**: 修改 `MA_PERIODS`
- **输出路径**: 修改 `OUTPUT_DIR`
- **请求频率**: 修改 `REQUEST_DELAY`（默认 0.5s，避免限流）

## 分析逻辑说明

### 自选股多空评分体系

程序综合以下维度给出操作参考建议：

1. **均线排列**（权重最高）：多头排列 → 看多，空头排列 → 看空
2. **价格 vs MA20**：上方 → 偏多，下方 → 偏空
3. **近 3 日涨跌幅**：连续大涨 → 短线偏强但防回调，连续大跌 → 超跌需谨慎
4. **MACD 金叉/死叉**：金叉 → 偏多信号，死叉 → 偏空信号
5. **DIF 位置**：零轴上方 → 强势区，零轴下方 → 弱势区

### 操作建议等级

| 多空强度比 | 建议 |
|-----------|------|
| ≥ 70% 多头 | 🟢 偏多 — 持有观察，轻仓关注回调机会 |
| 55%–70% | 🟡 偏多震荡 — 建议持有观望 |
| 40%–55% | 🟠 方向不明 — 观望为主 |
| 25%–40% | 🟡 偏空震荡 — 重仓关注减仓机会 |
| < 25% | 🔴 偏空 — 控制仓位，注意风险 |

## 注意事项

1. **交易时间**：建议在交易日 15:00 后运行以获取完整日线数据；盘中运行也能获取实时数据
2. **网络要求**：需要能访问东方财富等公开行情接口；如遇超时，程序会自动重试 3 次
3. **限流保护**：每次请求间隔 0.5 秒，放心使用不会封 IP
4. **首次运行**：akshare 会下载一些缓存数据，稍慢是正常的
5. **非交易日**：周末/节假日运行可能数据为空或为上一交易日数据

## 常见问题

**Q: 运行报错 `ModuleNotFoundError: No module named 'akshare'`**
A: 执行 `pip install akshare pandas numpy` 安装依赖

**Q: akshare 接口报错或返回空数据**
A: 可能是东方财富接口临时波动，等待几分钟重试即可；程序已内置 3 次自动重试

**Q: HTML 报告中文乱码**
A: 用 Chrome/Edge 浏览器打开，报告使用 UTF-8 编码，不要用记事本打开

**Q: 如何添加更多自选股**
A: 编辑 `config.py` 中 `WATCH_STOCKS` 字典，按格式添加即可

**Q: 可以设置定时运行吗**
A: Windows 可用"任务计划程序"、Linux/macOS 可用 `cron`，设置 `python main.py` 定时执行
